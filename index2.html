Cours jQuery - Traverser le DOM

Un paragraphe HTML

Quelques méthodes jQuery permettant de remonter le DOM :

  • parent()
  • parents()
  • parentsUntil()
  • closest()

Quelques méthodes permettant de descendre le DOM :

  • children()
  • find()

Plus de méthodes pour traverser le DOM :

  • siblings()
  • next()
  • nextAll()
  • nextUntil()
  • prev()
  • prevAll()
  • prevUntil()

Ces méthodes servent à accéder aux éléments frères d'une première collection d'éléments.
